# Event Booking System

Final group project for CPAN-228 Web Application Development.

Technologies
- Spring Boot
- Thymeleaf
- Spring Data JPA
- H2 Database
- Bootstrap

How to Run
1. Clone repository
2. Run: mvnw.cmd spring-boot:run
3. Open: http://localhost:8081
