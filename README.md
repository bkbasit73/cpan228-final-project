## 📖 Project Description
This is a web-based Event Booking System developed using:

- Spring Boot
- Thymeleaf
- Spring Data JPA
- H2 Database
- Bootstrap

Users can create and manage events such as workshops, meetups, seminars, and conferences.

---

## 🚀 Features Implemented

### ✔ Informational Pages
- Home Page
- About Page
- Contact Page

### ✔ Event Management
- Add new events
- View all events
- Auto-loaded sample events

### ✔ Data Validation
- Required fields validation
- Capacity limits
- Price validation

### ✔ Filtering & Sorting
Users can:
- Filter by event type
- Sort by:
  - Date
  - Capacity
  - Price

### ✔ Database
- H2 in-memory database
- Data persists during runtime
- Sample data loads automatically on startup

### ✔ UI
- Responsive Bootstrap design
